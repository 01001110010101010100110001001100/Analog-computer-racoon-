# Analog-computer-racoon-
Bear bones analog function generetor with display

This is a basic analog circuit that runs on 9 v bateries and thake a steady dc voltage and transformes it into a 3D display of a parametric function based on this project :
https://www.desmos.com/calculator/f68b1ee804 Warning the desmos file requiers an external script as the total nr of calculations for 1 point exede the cap of 10.000

The updated mathemathical functions will be changed from Gaussian Bell Curves to sin and cosin sums as a byproduct of optimazing for the Fourie Transform .
